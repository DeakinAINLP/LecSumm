Dimensionality The size of the dimensions is the number of attributes. The dimensions captured for the various data types will be different. For example, image data, the dimensions for a 64x64 image would be 4,096 dimensions. This would also be an example of high dimensional data.  Curse of dimensionality As the dimensionality increases, the volume of the space increases which causes the data to become sparse. Sparsity in data refers to a high volume of possible combinations remaining unobserved. The space between points in high-dimensional spaces becomes less useful because the value of the distance increase in uniformity.  Solving the cure Dimensionality reduction is the process of converting a set of data that has vast dimensions into  a  data  set  with  fewer  dimensions.  For  example,  if  the  data  is  numerical  and  there  are duplicate features, redundancy can be removed by only using one of the features to represent all duplicates.
