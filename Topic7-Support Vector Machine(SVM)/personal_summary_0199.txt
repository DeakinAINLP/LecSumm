In this topic we covered Linear models including linear and logistic models used for both regression and classiﬁcation. Linear regression works by ﬁnding the line of best ﬁt for the training data using a single straight line ﬁttng all predicted data to this line. Logistic regression works in a similar way to linear regression, but it instead compares the data to a logit function created using the training data.
