The unit content in topic one was an introduction to machine learning concepts.   Received an overview of the types of machine learning, and the ways machine learning can be applied. For example, computer vision, board games, etc. Introduced to the steps in machine learning and an overview of what is performed in each step. The three steps discussed were Data Manipulation, Analytics, and Evaluation and Visualisation. Introduced to the three categories of machine learning. Supervised Learning, Unsupervised Learning, and Reinforcement Learning.     Review of the mathematical concepts needed for machine learning. For example, vector and matrix operations.    Started using Python to complete programming tasks.
