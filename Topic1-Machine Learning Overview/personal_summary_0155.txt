This topic consisted of an introductory approach to the concepts of machine learning with the main points being:  1. Definitions of machine learning. 2. Real world applications of machine learning 3. Steps in the machine learning process 4. Differences between supervised and unsupervised learning. 5. Evaluation and selection of machine learning models. 6. Mathematical implementations in the field of machine learning using vector operations and matrix  algebra.  7. Use of python and python packages to implement arithmetic operations in machine learning.
